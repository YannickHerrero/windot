# Firefox Extensions

## Proton Pass

Password manager from Proton.

**Install:** [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/proton-pass/)

## uBlock Origin

Efficient ad and tracker blocker.

**Install:** [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/)

## Tridactyl

Vim-like keyboard navigation for Firefox. Enables modal editing, custom keybindings, and command-line interface.

**Install:** [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/tridactyl-vim/)

**GitHub:** https://github.com/tridactyl/tridactyl

**Tips:**
- Press `:` to open command mode
- Use `f` to show link hints for clicking
- Use `j/k` to scroll down/up
- Use `d` to close tab, `u` to undo close
- Run `:tutor` for an interactive tutorial
